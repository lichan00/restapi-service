Project Name: RestApi-Service
Description: This is a project that connects to a URL, parses the data and displays the response.

Prerequisite: 
   Gradle 3.4.1 above
   Java 8

How to:
   To build the project, go to the source directory and execute the command ./gradlew build
   The output (restapi-service-0.0.j-SNAPSHOT jar file) is located at $src/build/libs and can be run using
     java -jar command
